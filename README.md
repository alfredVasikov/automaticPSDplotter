This MATLAB script processes experimental particle size data to analyze and visualize Particle Size Distributions. It automates data cleaning, statistical metrics extraction, and individual distribution plotting.Regarding technical capabilities, the script extracts critical distribution percentiles including d10 for fine particles, d50 for median size, and d90 for coarse particles to define sample boundaries. It also computes the Sauter Mean Diameter (d32) based on the surface area to volume ratio for fluid dynamics and heat transfer applications. The tool automates data cleaning by filtering out invalid data points, completely stripping away missing values and non-positive matrix entries. For visualization, it generates individual Probability Density Function histogram plots using customized size bins from zero to sixty micrometers. Finally, it provides integrated reporting by embedding the full statistical metrics summary directly into the legend box of each respective figure.The required input file is named input_psd.xlsx. This Excel spreadsheet must contain a raw data matrix where each column represents independent particle measurement samples for a distinct specimen.
